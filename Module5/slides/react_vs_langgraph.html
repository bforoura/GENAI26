Defining the State

The key to parallelism is the Reducer, which prevents data from being overwritten.

import operator
from typing import Annotated, TypedDict

class GraphState(TypedDict):
    # Original User Input is stored here
    user_query: str
    # Results APPEND instead of REPLACE
    results: Annotated[list[str], operator.add]
    tool_queries: list[str]
    final_answer: str
