How are Embeddings Determined?

We can use a Gemini embedding model which already contains normalized embedded vectors.

The Normalization Step

Vectors are scaled to a "unit length" of 1 so that search results are based purely on topic similarity rather than text length.

Vector DB Operations

  • update() or delete() records using their unique IDs.
  • Store and filter by extra data; e.g., source or date, during retrieval.
  • peek() at data, count() items, or delete entire collections.

LangChain Implementation

In a real app, you would use these functions to move data in and out:

# Create the vector store:
vector_store = Chroma(embedding_function=embeddings)

# To Save/Index Chunks:
vector_store.add_documents(docs)

# To Retrieve Chunks:
results = vector_store.similarity_search( "How do language models work?", k=2 )
